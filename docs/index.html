A complete, customisable
course catalog for Moodle.

A free plugin that adds a filterable, searchable course catalog to your Moodle site. Custom field filters, tag search, live AJAX results.

Open-source · GPL v3 · English and French included

Download v1.0.1 GitHub

Requires Moodle 4.3+ and PHP 8.3+

Features

Custom field filters

Text, select, checkbox, number, textarea — with AJAX autocomplete on text fields.

Tag search

Pill selector with autocomplete and popular tag suggestions.

Live results

Results update as you filter — no page reload.

Configurable order

Set the display position of every filter independently.

Guest access

Let non-logged-in users browse the catalog and preview courses.

Accessibility

Full keyboard navigation in all dropdowns, ARIA labels on all inputs.

Installation

1
Download local_catalog_browser.zip using the button above.
2
In Moodle, go to Site administration → Plugins → Install plugins and upload the ZIP.
3
Follow the on-screen prompts to complete the installation.
4
Go to Site administration → Plugins → Local plugins → Catalog Browser to configure everything. The catalog URL is shown at the top of the settings page.
